# Socket.io Chat V2

A new version of my simple chat with more features :
- Username and room choice
- Username and time of messages
- Current room name
- Users list
- See who is typing
- Different background for messages

## Run Locally

Install dependencies

```bash
  npm install express socket.io
```

Start the server

```bash
  npm run start
```

Link in browser

```bash
  http://localhost:3000/
```
